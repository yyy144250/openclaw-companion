# 2026-03-12 Madoka 重构

## 重大重构：从单窗口应用改为双窗口桌面看板娘

### 改动概要
- **架构**: 从单窗口 450x700 普通应用 → 双窗口（mascot 透明拖拽 + chat 弹出聊天面板）
- **Tauri 配置**: 启用 `transparent: true`, `decorations: false`, `alwaysOnTop: true`, `skipTaskbar: true`, `macOSPrivateApi: true`
- **Rust 后端**: 添加 `open_chat_window`, `close_chat_window`, `toggle_chat_window`, `start_dragging`, `set_mascot_ignore_cursor` 命令
- **前端路由**: App.tsx 根据 `getCurrentWebviewWindow().label` 区分 mascot/chat 视图
- **连接方式**: 从手动填 `ws://ip:port` 改为只输入 host，自动探测端口（probeServer）
- **首次设置**: 新增 SetupWizard 组件，首次启动引导用户配置
- **图片消息**: ChatInput 支持粘贴/拖拽/选择文件发送图片（base64），ChatMessages 支持显示图片
- **UI 主题**: 全面暗色毛玻璃设计
- **状态管理**: AppSettings 新增 `server: ServerConfig`（host/port/secure/token），版本迁移到 v4

### 新增文件
- `src/views/MascotView.tsx` + `.css` — 看板娘桌面视图
- `src/views/ChatView.tsx` + `.css` — 聊天面板视图  
- `src/components/SetupWizard.tsx` + `.css` — 首次配置向导
